## Overview

The **AI-Driven Finance Dashboard** is designed to help users analyze and visualize their financial transactions. It leverages AI for:  

- Predicting future spending trends  
- Detecting anomalies or unusual spending  
- Generating interactive charts for better insights  

The dashboard can also export processed data for further visualization in **Power BI**.  

---

## Features

- **Transaction Overview:** View total spending by date and category  
- **AI Predictions:** Forecast spending trends for the next 30 days  
- **Anomaly Detection:** Identify unusual or unexpected transactions  
- **Interactive Charts:** Visualize data dynamically with Plotly  
- **Power BI Integration:** Export data for advanced dashboards  

---

## Technologies

- **Python** – Data processing and AI models  
- **Flask** – Backend server and API  
- **Plotly** – Interactive charts and visualizations  
- **scikit-learn** – Machine learning models for prediction and anomaly detection  
- **Power BI** – Optional for advanced dashboards
